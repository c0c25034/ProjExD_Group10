# オセロ

## 実行環境の必要条件
- python >= 3.10
- pygame >= 2.1

## ゲームの概要
- 通常のオセロに、盤面拡大・ランダム爆破・スキル機能などを追加した対戦型ボードゲーム。
- 参考URL：[【Pygame】オセロゲームを作成する方法](https://python.joho.info/pygame/pygame-othello/)

## ゲームの遊び方
- 黒と白の石を交互に置き、相手の石を挟むことで自分の石に変える。
- 最終的に盤面上の石が多いプレイヤーの勝利。
- スキルポイントを消費することで特殊スキルを使用できる。

## ゲームの実装
### 共通基本機能
- 描画
- 盤面の描画
- 石の描画
- 石を置ける場所の判定
- 石をひっくり返す処理
- ターン管理
- 勝敗判定

### 分担追加機能
- 盤面拡大機能(担当：赤岡)：一定ターン経過後、盤面が8×8から10×10に拡大する。
- ランダム爆破機能(担当：大沢)：コマが消える。石をおいたとき一定の確率で起こる。
- スキルポイント(担当：)：一度に多くの石を取るほどスキルポイントが多く溜まる。
- スキル機能(担当：櫻井)：スキルポイントを消費して、爆破・マスの一時無効化・角の取得などの特殊効果を発動できる。
- エフェクト機能(担当：高岡)：石をひっくり返すときや爆破時に視覚的な演出を表示する。
- ガイド機能(担当：寺地)：現在のプレイヤーが石を置ける場所を白い半透明の円で表示する機能

### メモ
- クラス内の関数はすべてguideで始めるように設計してある
- できるだけ講義資料やこうかとんのゲームで使われているメソッドを使用した
- コード規約に注意を払った
- クラス内で用いられている関数はすべてクラス外で定義した
